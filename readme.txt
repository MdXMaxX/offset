usage: offset.py [-h] [--offset OFFSET] [--ignore {ssc,sm}] [--noprompt] [directory [directory ...]]

adds the specified offset (in seconds, default +0.009) to all .sm and .ssc files found in the specified directories
and their subdirectories

positional arguments:
  directory

optional arguments:
  -h, --help         show this help message and exit
  --offset OFFSET
  --ignore {ssc,sm}
  --noprompt
